#Simple Sentiment Analysis API

An end-to-end AI project that analyzes the sentiment of text using Natural Language Processing (NLP). 

This project features a **REST API** built with FastAPI and is **deployed** to a cloud environment for public access.

## 🚀 Features
* **AI Engine:** Powered by `TextBlob` for polarity analysis.
* **Backend:** `FastAPI` for high performance and automatic Swagger documentation.
* **Deployment:** Production-ready on Render.
* **Input:** Accepts English text strings.
* **Output:** Returns Sentiment labels (Positive, Negative, Neutral) and a precise confidence score.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Framework:** FastAPI
* **NLP Library:** TextBlob
* **Server:** Uvicorn
